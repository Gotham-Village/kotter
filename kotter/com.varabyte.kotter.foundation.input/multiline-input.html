multilineInput

fun MainRenderScope.multilineInput(initialText: String = "", id: Any = Unit, viewMap: ViewMapScope.() -> Char? = null, customFormat: CustomFormatScope.() -> Unit? = null, pageSize: Int? = null, isActive: Boolean = true)

Like input, but allows the user to type multiple lines of text.

To end a multiline input string, the user must press Ctrl+D, as pressing Enter just appends a new line.

Parameters

pageSize

How many lines of text to skip over when pressing Page Up / Page Down. If not specified, uses the value defaultPageSize, which you can set instead to affect all inputs.
