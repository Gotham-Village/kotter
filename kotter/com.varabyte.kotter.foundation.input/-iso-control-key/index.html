class IsoControlKey : Key

Marks a key as one that triggers behavior (as opposed to being a simple typeable value).

This has nothing to do specifically with the Ctrl key on the keyboard. For example, Keys.ENTER, Keys.BACKSPACE, and Keys.UP are control keys.

See also Character.isISOControl
